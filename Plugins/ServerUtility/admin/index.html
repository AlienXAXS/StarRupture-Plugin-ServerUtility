ServerUtility Admin Panel

This is the admin API for the ServerUtility dedicated server plugin. All endpoints require POST requests with a JSON body. Authentication is session-token based.

Authentication

POST /ServerUtility/api/auth/login

Exchange your API key for a session token. Tokens expire after the configured SessionExpiry (default 3600 s).

FieldTypeDescription
api_keystringThe key configured in ServerUtility.ini → [AdminPanel] ApiKey
// Request
{ "api_key": "your-secret-key" }

// Response
{ "ok": true, "session_token": "a3f9..." }

Player Management

POST /ServerUtility/api/players/list

List all currently connected players.

{ "session_token": "..." }
// → { "ok": true, "players": [{ "index": 0, "name": "Alice", "ping": 42, "ip": "1.2.3.4" }] }

POST /ServerUtility/api/players/kick

FieldTypeDescription
session_tokenstringActive session token
player_indexintZero-based index from /players/list
{ "session_token": "...", "player_index": 0 }
// → { "ok": true }

POST /ServerUtility/api/players/ban

Kick the player and record their IP in Plugins/ServerUtility/bans.txt.

{ "session_token": "...", "player_index": 0 }
// → { "ok": true }

Wave Control

POST /ServerUtility/api/wave/status

{ "session_token": "..." }
// → { "ok": true, "in_progress": true, "paused": false, "type": 2, "stage": 1, "progress": 0.35 }

POST /ServerUtility/api/wave/start

FieldTypeDescription

{ "session_token": "...", "wave_type": 1 }

POST /ServerUtility/api/wave/cancel

{ "session_token": "..." }

POST /ServerUtility/api/wave/pause

{ "session_token": "..." }

POST /ServerUtility/api/wave/resume

{ "session_token": "..." }

Inventory

POST /ServerUtility/api/inventory/give

Give an item to a connected player by full Unreal Engine asset path.

FieldTypeDescription
player_indexintZero-based index from /players/list
asset_pathstringFull UE asset path, e.g. /Game/Items/Weapons/BP_Sword
countintNumber of items to give (default: 1)
{ "session_token": "...", "player_index": 0, "asset_path": "/Game/Items/Weapons/BP_Sword", "count": 5 }
// → { "ok": true }

Configuration (ServerUtility.ini)

[AdminPanel]
Enabled=true
ApiKey=your-secret-key-here
SessionExpiry=3600

All failed requests return { "ok": false, "error": "..." }. Unauthorized requests return HTTP 401. Non-POST requests to /api/ paths return HTTP 403.
